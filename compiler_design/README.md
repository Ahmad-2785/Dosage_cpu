### Dosage C Compiler
***
OverView : Generally a Compiler have 7 phases(khan)
including:
1. Pre Processing
2. Lexical Analyzer
3. Syntax Analyzer
4. Semantic Analyzer
5. Intermediate Code Generator 
6. Code Optimizer
7. Target Code Generator
